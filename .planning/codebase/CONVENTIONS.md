## English Version

# Conventions

### Code Style
- **HTML/CSS**: Emphasizes semantic HTML (`<main>`, `<section>`, `<article>`, `<header>`, `<nav>`) and responsive design using CSS Flexbox/Grid without external libraries.
- **C#**: Standard OOP naming conventions (PascalCase for classes and methods, camelCase for parameters).
- **SQL**: `tb_` prefix for tables, snake_case for fields.

### Accessibility
- Strong focus on WAI-ARIA (`aria-label`, `aria-live="polite"`, `role="progressbar"`).
- Support for high contrast and screen readers.

## Versão em Português

# Convenções

### Estilo de Código
- **HTML/CSS**: Ênfase em HTML semântico (`<main>`, `<section>`, `<article>`, `<header>`, `<nav>`) e design responsivo utilizando Flexbox/Grid sem bibliotecas externas.
- **C#**: Padrões de nomenclatura POO tradicionais (PascalCase para classes e métodos, camelCase para parâmetros).
- **SQL**: Prefixo `tb_` para tabelas, padrão snake_case para campos.

### Acessibilidade
- Foco intenso em WAI-ARIA (`aria-label`, `aria-live="polite"`, `role="progressbar"`).
- Suporte para modos de alto contraste e leitores de tela nativos.
