# Tech Stack

This directory contains the practical MVP code for **Nex_TI**.

### Current State
- **Frontend**: Semantic HTML5, CSS3, Vanilla JavaScript.
- **Backend**: C# .NET 10 Minimal APIs, Entity Framework Core.
- **Database**: SQL Server (DDL scripts).
- **Visualization**: Neural Map engine with spring physics simulation (`neural-map.js`).
